# BOLVEDA

AI-powered knowledge assistant for contextual document question answering using RAG.

## Features
- PDF ingestion
- Semantic search
- ChromaDB vector storage
- Context-aware retrieval
- Grounded LLM responses
- Voice input/output

## Tech Stack
- Python
- LangChain
- ChromaDB
- OpenAI
- Streamlit
- Whisper
- ElevenLabs
